# Logistics Operation Analytics 🚚
## Introduction 🧾
![logistics overview](assets/images/capstone_dashboards_5.png)
*A comprehensive view of logistics operations and safety compliance.*

These dashboards were created as part of a capstone project to provide comprehensive overview of logistics operations in a company by monitoring financial performance, customer service and delivery efficiency, fleet operations, fuel performance and asset maintanance, driver performance and safety compliance. It enables stakeholders track important operational metrics,identify performance gaps, cut down operational costs, optimize fleet assets and further make future data drivrn decisions. The dataset for this project was provided by Ts Academy with data from the past three years.
